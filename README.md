# 🎥 YouTube Video Downloader

ایک آسان YouTube video downloader جو مختلف quality میں videos download کرنے کی سہولت فراہم کرتا ہے۔

## ✨ Features

- ✅ کسی بھی YouTube video کو download کریں
- ✅ مختلف quality options (Highest, 1080p, 720p, 480p, 360p)
- ✅ Simple اور user-friendly interface
- ✅ Fast downloads
- ✅ Video information display

## 🚀 Installation

### Step 1: Dependencies Install کریں

```bash
npm install
```

### Step 2: Server Start کریں

```bash
npm start
```

### Step 3: Browser میں Open کریں

```
http://localhost:3000
```

## 📖 استعمال کا طریقہ

1. **YouTube URL paste کریں**: کسی بھی YouTube video کا URL copy کر کے input field میں paste کریں
2. **Quality select کریں**: اپنی پسند کی quality choose کریں (Highest, 1080p, 720p, 480p, 360p)
3. **Download button دبائیں**: "Download Video" button پر click کریں
4. **Video download ہو جائے گی**: آپ کی video automatically download ہونا شروع ہو جائے گی

## 🎯 Quality Options

- **Highest**: بہترین available quality میں download کریں
- **1080p**: Full HD quality
- **720p**: HD quality
- **480p**: Standard quality
- **360p**: Low quality (کم data استعمال کرتا ہے)

## 🛠️ Technologies استعمال شدہ

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **YouTube API**: ytdl-core

## ⚙️ Development Mode

اگر آپ development کے دوران automatic restart چاہتے ہیں:

```bash
npm run dev
```
